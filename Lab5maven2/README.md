# Переделка 5 лабораторной под сборку в maven

Для использования программы, важно указать переменную окружения "FILE_PATH" при помощи команды: 
> setx "FILE_PATH" "*Полный_путь_до_лабораторной*"


- Переменной окружения можно указать любое удобное для вас значение, однако, для удобства проверки работоспособности команд и коллекции, можно указать путь конкретно до лабораторной, т.к. там уже присутствуют нужные для этого файлы. 
- Для сборки рекомендуется использовать команду "mvn clean package" внутри папки с лабораторной. 
- Очевидно, для запуска программы стоит использовать файл .jar, использующий все зависимости.
